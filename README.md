
Aplicação web desenvolvida com React + Vite + TypeScript e Bootstrap via CDN para a disciplina de Desenvolvimento de Software WEB (Prof. Alexandre Almeida).
Permite gerenciar o estoque de materiais de crochê (fios, agulhas e acessórios), acompanhar os níveis de estoque em tempo real e registrar entradas e saídas de material.



npm install
npm run dev
